# Dream Catcher Overlay

Hypixel Bedwars overlay that displays opponents' FKDR and star after /who command is performed in game.
Displays players of varying threats in different colours based on their FKDR/star

# Features
Whitelist (to ignore your own party members)
Ability to use own Hypixel Developer API key
